## 关于这套体系

### 核心思想
- **能量是动力，信息是能量操作的输出**
- 任何系统都可以用 6 个元单元 + 5 类功能单元 + 劳动点 + 显像器 描述
- 复合单元用 → 和 ↹ 表达职责偏向
- 扩展缀（绝对/相对/单向/超复杂）描述极端情况
- **原件/组织**让体系可以谈论自己
